What actually happens

Follow the prompt through the system — your knowledge lights up at every step.

Big Data Engineering
The prompt hits a load-balanced API cluster. The model's 100GB+ weights are served from distributed GPU memory — the same infrastructure patterns you studied.
Natural Language Processing
The sentence is tokenised — broken into subword units. "Quantum" and "computing" become tokens, each mapped to a position. The preprocessing pipeline you built in your NLP module runs here.
Vector Space & Embeddings
Each token is converted into a high-dimensional vector — a coordinate in meaning space. "Quantum" lands near "physics" and "superposition", far from "recipes". The embedding layer you studied is doing this.
Neural Networks
The vectors pass through dozens of transformer layers — attention heads, feed-forward networks, residual connections. Every concept you covered in the neural networks module is firing simultaneously.
KNN & Similarity
The model identifies the nearest contextual neighbours for "explain" + "simply" — navigating toward accessible language rather than academic jargon. Proximity in vector space guides every word choice.
Unsupervised Learning
All of this structure — the clusters, the semantic neighbourhoods, the model's knowledge — emerged without human labels. Billions of self-supervised predictions on raw text created the map.
Supervised Learning & RLHF
The word "simply" triggers alignment training. The model has learned, through labelled fine-tuning and human feedback rankings, that this prompt expects a clear, accessible response — not a research paper.
LLM — The Convergence
Token by token, the response appears. Every word is a prediction — informed by data pipelines, shaped by neural architecture, located in vector space, navigated by similarity, refined by alignment. One prompt. All eight modules. Simultaneously.
